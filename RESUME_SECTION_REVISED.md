# Revised Resume Section - ATS Optimized

**VirtuTrade | Swift, SwiftUI, Combine, Core Data, MVVM, REST APIs** | Nov 2024 – Dec 2024

• Developed a native iOS cryptocurrency tracking application tracking 250+ cryptocurrencies with real-time market data integration, interactive 7-day price charts using iOS Charts framework, and virtual portfolio management with Core Data persistence for CRUD operations and automatic data synchronization.

• Architected reactive data flow using Combine framework with publisher-subscriber pattern, integrating RESTful API endpoints (CoinGecko API v3) via URLSession with robust error handling, automatic retry logic (3 attempts), HTTP response validation, JSON decoding with Codable protocols, and debounced search (0.5s) to minimize API calls.

• Implemented MVVM architecture with centralized state management, optimized performance through lazy loading (LazyHStack, LazyVGrid), local image caching with FileManager, and designed modern SwiftUI interface with custom theme system supporting light/dark modes, haptic feedback, NavigationStack, and responsive layouts for all iOS device sizes.
